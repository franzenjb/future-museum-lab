# Future Museum Lab

A standalone interactive companion to a white paper about the future of museums.

This is not a PMA website redesign and not a collection explorer. It is a speculative digital exhibition about the idea that the future art museum is a civic operating system.

## Routes

- `/` - Future Museum Lab landing experience
- `/paper` - formal white paper
- `/outline` - structured argument outline
- `/lab` - speculative demo index
- `/conversation` - preserved conversation fragments
- `/slow-looking` - client-side Slow Looking Room mini-app

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verification

```bash
npm run lint
npm run build
npx playwright test
```
